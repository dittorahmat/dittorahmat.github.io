# Tech Context

## Technologies Used:
- Hugo: Static site generator
- Markdown: Content format
- HTML, CSS, JavaScript: Web technologies for the theme
- GitHub Pages: Hosting platform
- Git: Version control

## Development Setup:
- Local Hugo installation
- VSCode or preferred text editor
- Git CLI

## Technical Constraints:
- GitHub Pages limitations (e.g., build times, custom domains)
- Hugo theme customization limitations

## Dependencies:
- Hugo CLI
- Git
